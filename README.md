# BewerbungsCheck

Kostenloses Tool zur Optimierung von Bewerbungsunterlagen. Lebenslauf und Stellenanzeige eingeben – sofort ATS-Score, fehlende Keywords und Verbesserungsvorschläge erhalten.

**→ [wirsig.github.io/bewerbungscheck](https://wirsig.github.io/bewerbungscheck)** *(URL anpassen)*

-----

## Features

- **ATS-Score** – wie gut passt dein Lebenslauf zur Stelle (0–100%)
- **Keyword-Analyse** – gefundene und fehlende Keywords auf einen Blick
- **Verbesserungsvorschläge** – konkrete Hinweise zu Struktur, Länge und Inhalt
- **Anschreiben-Checkliste** – 12 Punkte vor dem Absenden abhaken
- **100% lokal** – keine Daten verlassen deinen Browser, kein Server, kein Tracking

## Dateien

```
index.html        → Das Tool
impressum.html    → Impressum
datenschutz.html  → Datenschutzerklärung
README.md         → Diese Datei
```

## Technologie

Reines HTML/CSS/JavaScript – kein Framework, kein Backend, keine externe API.  
Schriften via [Google Fonts](https://fonts.google.com) (Syne + DM Sans).
